Estudios de Caso

COMENTAD: Diseño y Desarrollo

En este proyecto, trabajé a base de una situación para mejorar la comunicación entre estudiantes y profesores universitarios, por lo que se hizo un gran trabajo de investigación donde se hicieron entrevistas, desk research, encuestas y más para ver que era lo que más querían los estudiatnes, luego de eso, se hizo el proceso de diseño con Arquiectura de infromación, prototipos en papel, wireframes y prototipos para poder seguir a la parte de desarrollo. En la parte de desarrollo utilicé Android Studio y Kotlin para el desarrollo de la aplicación móvil y Java Script para el desarrollo web. Uno de los principales desafíos fue la integración de las dos aplicaciones, lo que resolvimos con la integración de Firebase.

ZONAR: Desarrollo Experiencia de Usuario

El desarrollo de ZONAR con mis compañeros, se centro en una experiencia presencial accesible donde los usuarios podían crear sus propios secuencias musicales a base de unos cubos en los que cada uno sacaba una música diferente. Para eso se necesitó de gran investigación de usuarios, entrevistas y talleres acerca de la discapacidad visual para un proyecto bien estructurado. Utilicé Arduino para utilizar sensores que identificaban cual cubo se estaba utilizando y con mis compañeros Unity y Fmod para implementarlos a la música.
